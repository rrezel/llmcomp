---
name: feedback_no_unsolicited_changes
description: Never make code changes without being asked. Answer questions first, then wait for instructions.
type: feedback
---

Do not make code changes unless explicitly asked. When the user asks a question or points out a problem, answer the question or acknowledge the problem first. Wait for the user to tell you what to do. Do not assume the next step.

**Why:** The user has repeatedly been frustrated by premature code changes that went in the wrong direction, wasting time and creating more problems to fix.

**How to apply:** When the user says something is wrong, diagnose it first. Show your understanding. Wait for confirmation before writing any code.
